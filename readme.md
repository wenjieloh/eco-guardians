# 🌿 EcoGuardians: Invasive Plant Alert

An interactive educational platform designed for secondary and primary school students to learn about invasive plant species, their impact on local habitats, and how to take action.

## 🎯 Project Goals
* **Educate:** Teach students how to identify high-risk invasive species like Kudzu and Giant Hogweed.
* **Engage:** Use gamification (Memory Match) and interactive quizzes to reinforce learning.
* **Empower:** Provide clear, actionable steps for students to become "Citizen Scientists."

## 🛠️ Features
* **Interactive Habitat Cards:** Dynamic profiles of invasive plants with danger-level color coding.
* **Memory Match Game:** A custom JavaScript-powered game to help with visual identification.
* **Animated Danger Alerts:** CSS-driven pulse animations to highlight hazardous species.
* **Quick Assessment:** A built-in quiz to test knowledge retention.
* **Responsive Design:** Optimized for both desktop and iPad/tablet use.

## 💻 Tech Stack
* **HTML5:** Semantic structure for accessibility.
* **CSS3:** Custom styling with CSS variables and keyframe animations.
* **JavaScript (Vanilla):** Dynamic DOM manipulation and game logic.
